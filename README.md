# LATT v4.0.0 [Guardian] - Local AI Tiny Trust Protocol

### 核心定位
本仓库是 LATT 协议的第四代正式版本。
- **L**everage: 利用本地物理算力 (GTX 1050 Ti)
- **A**I Enhance: DeepSeek-R1 极限内核
- **T**rim: 彻底剥离 OpenClaw/Node_modules 冗余
- **T**arget: 极小众本地审计与算力主权需求

### 快速部署
1. `chmod +x deploy.sh`
2. `./deploy.sh`
3. `python3 core/main.py`
